# Contributing

Contributions are welcome! Here's how you can help:

## Getting Started

1. Fork this repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Run the app locally: `streamlit run app.py`
5. Commit your changes: `git commit -m "feat: your feature"`
6. Push to your branch: `git push origin feature/your-feature`
7. Open a Pull Request

## Ideas for Contributions

- Add more product catalogs (optics, electronics, etc.)
- Improve the search/matching algorithm
- Add more LLM providers to the routing table
- Improve the PDF export layout
- Add unit tests
- Add internationalization (i18n) support

## Code Style

- Python 3.10+
- Use type hints where practical
- Keep functions small and focused
- Follow existing patterns in the codebase

## Issues

Found a bug? Have a feature request? Please [open an issue](../../issues).
